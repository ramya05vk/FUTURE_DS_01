# 📊 Business Sales Performance Dashboard

## 🚀 Project Overview

This project analyzes real-world e-commerce transaction data to uncover key business insights.
The workflow includes **data cleaning, analysis, and interactive dashboard creation** using Python and Tableau.

---

## 🎯 Objectives

* Analyze sales performance over time
* Identify top-performing products
* Understand regional sales distribution
* Detect high-value customers
* Build an interactive business dashboard

---

## 🛠️ Tools & Technologies

* **Python (Pandas, Matplotlib)** – Data cleaning & analysis
* **Tableau** – Interactive dashboard visualization
* **CSV Dataset** – Real-world retail transaction data

---

## 🔄 Project Workflow

### 1. Data Cleaning

* Removed missing values (CustomerID)
* Filtered cancelled transactions
* Removed negative quantities and invalid prices
* Converted date columns to proper format
* Created Revenue column (Quantity × UnitPrice)

### 2. Data Analysis

* Monthly revenue trend analysis
* Top-selling product identification
* Country-wise sales performance
* High-value customer analysis

### 3. Dashboard Development

* Built an interactive Tableau dashboard with:

  * Revenue trend visualization
  * Top products analysis
  * Country-level sales map
  * Top customers chart
  * Key business insights

---

## 📈 Key Insights

* Revenue shows an upward trend with peak performance in later months
* A small group of products contributes significantly to total revenue
* The United Kingdom dominates overall sales performance
* High-value customers generate a major portion of revenue
